Chapter 48 Reshaping

48.1 Stack

tuples = list(zip(*[['bar', 'bar', 'baz', 'baz', 
                     'foo', 'foo', 'qux', 'qux'],
                    ['one', 'two', 'one', 'two',
                     'one', 'two', 'one', 'two']])) 
index = pd.MultiIndex.from_tuples(tuples, names=['first', 'second'])
df = pd.DataFrame(np.random.randn(8, 2), index=index, columns=['A', 'B'])
df2 = df[:4]
df2

The stack() method “compresses” a level in the DataFrame’s columns.

stacked = df2.stack()
stacked

With a “stacked” DataFrame or Series (having a MultiIndex as the index), the inverse operation of stack() is unstack(), which by default unstacks the last level:

stacked.unstack()
stacked.unstack(1)
stacked.unstack(0)

## Creating New Columns
df['new'] = df['A'] + df['B']
df

48.2 Removing Columns

df.drop('new', axis='columns') # could also use axis = 1
df # new is still there!!

48.3 Remove/Drop data (For real)

Use inplace to remove PERMENANTLY

df.drop('new', axis='columns', inplace=True)

48.4 Dropping rows

Default drop is row (or use axis = 0)

df.drop('second', axis='rows')
